# Game of Life

模擬細胞結構：

-  **Block**：靜止不動的穩定型態（Still Life）  
-  **Blinker**：原地振盪的週期型態（Oscillator）  
-  **Glider**：能在空間中移動的移動型態（Spaceship）  

會隨機生成多組不同型態的生命體。
